### A Carousel Plugin for Mobile Phone

***

##### Usage

```js
Carousel({
  el: '#el',      // CSS Selector OR HTMLElement
  time: 500,      // animation time
  autoPlay: 2000, // autoplay and set it's time
  imgs: [         // array of imgs's path
    'img/1.jpg',
    'img/2.jpg',
    'img/3.jpg'
  ]
})
```
